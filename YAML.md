# What is Yaml?

- Is used to represent data
- All the identation matters
- Yaml does not accept tabs, only spaces
- It must have an equal number of spaces

```yaml
Servers:
  - name: Server1
    owner: John
    created: 20200803
    status: active
```

## YAML

### Key Value Pair

```yaml
Fruit: Apple
Vegetable: Carrot
Liquid: Water
Meat: Chicken
```

### Array/Lists

```yaml
Fruits:
  - Apple
  - Banana
  - Orange

Vegetables:
  - Carrot
  - Cauliflower
  - Tomato
```

### Dictionary/Map

```yaml
Banana:
  Calories: 105
  Fat: 0.4 g
  Carbs: 27 g

Grapes:
  Calories: 62
  Fat: 0.3 g
  Carbs: 16 g
```
